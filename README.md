# Brewfile for @koni

## 利用方法


```bash

# homebrew をインストール
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

# このファイルをDL
curl -L https://raw.githubusercontent.com/t0m0120/Brewfile/master/Brewfile > ./Brewfile

# インストール
brew bundle
```
